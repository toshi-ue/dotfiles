# My dotfiles

## スクリプトの実行

```bash
bash -c "$( curl -fsSL https://raw.github.com/toshi-ue/dotfiles/main/setup.sh )"
```

## 手動インストール

- Google日本語入力
- Karabiner-ElementViewer
- BetterTouchTool
- DisplayLink Manager
